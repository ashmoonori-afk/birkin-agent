# Web Research

## When to Use

Use this skill when the task references external repositories, docs, standards, or facts that may have changed.

## Procedure

1. Prefer primary sources.
2. Record URLs and the specific files or pages consulted.
3. Separate source wording from interpretation.
4. Do not keep unsupported claims in deliverables.

## Pitfalls

- Do not rely on search snippets alone when a primary page is available.
- Do not copy large source passages.

## Verification

- Deliverables include a source note or reference file when external sources shaped the design.
